Twat

A CLI frontend to twitter, implemented in ruby.

Download as .zip Download as .tar.gz View on GitHub

Using Twat

twat is a cli interface to ruby.

A common example would be:

$ gem install twat
$ twat add [twitter handle]
$ twat "This is a tweet from me"
$ twat # This will put twat into follow mode
